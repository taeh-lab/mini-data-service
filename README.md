인테리어 AI 비서 설계도 - 인테리어 현장 사진 치수 추출 도구 – Ver1
1. 목적
현장 방문 없이
방 사진만으로 면적(평/㎡)을 확인하여
인테리어 작업 전 기초 치수 확보 시간을 줄인다.
2. 사용 대상
인테리어 실무자 1인 (내부 사용)
판단·견적·설명은 사람이 직접 수행
3. 입력
방 사진 1~3장
(정면 또는 전체가 보이는 사진)
4. 기능 범위 (Ver1)
수행 기능
사진을 기준으로 방 형태 단순화
면적 계산
출력 단위
㎡
평
5. 출력 결과 형식
코드 복사

방 면적
- 8.2㎡
- 2.48평
(시각적 참고용 간단한 방 형태 다이어그램 — 선택)

자재 계산
견적 산출
자동 판단 / 추천
메모, 노트, 저장 기능
주소 기반 처리

면적 값이 명확히 표시된다
